# Python `abs()`函数

> 原文： [https://thepythonguru.com/python-builtin-functions/abs/](https://thepythonguru.com/python-builtin-functions/abs/)

* * *

`abs()`函数返回数字的绝对值（大小无符号）。

其语法如下：

```py
abs(x) -> absolute value

```

| 参数 | 描述 |
| --- | --- |
| `x` | 任何数值 |

这是一个例子：

```py
>>> 
>>> abs(-45)
45
>>>
>>> 
>>> abs(-3.14)
3.14
>>> 
>>> 
>>> abs(10)
10
>>> 
>>> 
>>> abs(2+4j)
4.47213595499958
>>>

```

```py
print(abs(-45))
print(abs(-3.14))
print(abs(10))
print(abs(2+4j)) 
```

对于整数和浮点数，结果非常明显。 如果是`z = x + yi`的复数，则`abs()`函数将按如下方式计算绝对值：

绝对值= `|z| = √x² + y²`

```py
=> 2+4j
=> √2² + 4²
=> √4 + 16
=> √20
=> 2√5
=> 2*2.236
=> 4.472

```
