## 下载

### Docker

```
docker pull apachecn0/pythonguru-zh
docker run -tid -p <port>:80 apachecn0/pythonguru-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install pythonguru-zh
pythonguru-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pythonguru-zh
pythonguru-zh <port>
# 访问 http://localhost:{port} 查看文档
```
